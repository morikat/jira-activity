Jira-activity

JIRA-ACTIVITY


$ export JIRA_USER="hogehoge"
$ export JIRA_PASS="hogehoge_jira_password"
$ export JIRA_DOMAIN="jira.hogehoge.com"
$ export JIRA_ACTIVITY_MAX_RESULT=100
$ export JIRA_ACTIVITY_USER="hogehoge"

old format
$ jira-activity get
2015-02-13 PJ-1234 - bug fix
2015-02-12 PJ-1233 - hogehoge feature
2015-02-11 PJ-1232 - Coding

new(csv) format
$ jira-activity get csv
1,PJ-1234,bug fix,0.0,2015-02-13
1,PJ-1233,hogehoge feature,0.0,2015-02-12
1,PJ-1232,Coding,0.0,2015-02-11
